Detailed Description

implements CuControlsWriterFactoryI and creates Websocket writers Given a pointer to a Cumbia object and a CuDataListener, the create method instantiates and returns a CuWsWriter, that implements CuControlsWriterA interface. If setOptions is called before create, the options are set on the CuControlsWriterA.

CuWsWriterFactory can be cloned by means of the clone method.
