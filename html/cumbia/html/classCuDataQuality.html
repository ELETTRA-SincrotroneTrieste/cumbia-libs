Detailed Description

The CuDataQuality class maps quality descriptions that concern specific engines into an engine independent data quality information.

Engines that provide a quality for their data will use this class to make it engine independent

The set methods allow to OR multiple CuDataQuality::Type flags together in case quality needs describe multiple results. So, if quality represents the result of multiple readings, its type will have the CuDataQuality::Alarm flat set to true if one of the results is in alarm.
