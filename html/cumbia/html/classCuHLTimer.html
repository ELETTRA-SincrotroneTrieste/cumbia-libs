Detailed Description

The CuHLTimer class offers a higher level interface to CuTimer.

CuHLTimer timeouts are notified through a CuHLTimerListener Notifications carry an integer value representing the period (timeout). With this strategy, a single CuHLTimer can deliver timeouts at different periods, configured with addTimerListener.
