Detailed Description

service that implements a simple event loop for cumbia applications.

CuEventLoopService is a cumbia service (CuServiceI) that offers a simple event loop for an application. Please note that if you are developing a Qt application you should employ Qt's QApplication event loop (read CuThreadsEventBridge and Cumbia::registerActivity documentation).

See also the CuServiceProvider documentation.
