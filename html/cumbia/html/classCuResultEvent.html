Detailed Description

The CuResultEvent class stores a result that a CuActivity (sender) wants to deliver from the background to the main thread.

This is used internally by the library

The CuResultEvent is usually extracted in the CuThread::onEventPosted method. It is used to get the exchanged data and the pointer to the activity that sent the event
