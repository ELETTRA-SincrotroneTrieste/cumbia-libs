Detailed Description

interface for a concrete log implementation

This interface defines two write methods to write a message on a logging system (console, file, database, null...) and a getName method that compels every subclass to provide a unique name

The calls to CuLog::write are forwarded to every registered CuLogImplI implementation

See the CuLog::write documentation for details and CuConLogImpl::write for an example from an implementor.
