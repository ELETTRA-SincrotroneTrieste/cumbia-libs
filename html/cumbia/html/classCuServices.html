Detailed Description

This class provides an enumeration to define the available type of services available for the cumbia framework.

This class defines the type of available services for the cumbia framework. This is the list:

  • Thread the thread service: CuThreadService
  • ActivityManager the activity manager service: CuActivityManager
  • EventLoop a service implementing an event loop: CuEventLoopService
  • Log a logging facility: CuLog

Services are registered through the CuServiceProvider and must implement the CuServiceI interface, in order to be obtained back from the service provider after registration. User defined services complying with the CuServiceI interface can be registered.

See also CuServiceProvider::registerService, CuServiceProvider::unregisterService and CuServiceProvider::get.
