Detailed Description

an implementation of CuThreadsEventBridgeFactory_I for cumbia

This factory is used to instantiate a new CuThreadsEventBridge_I implementation, i.e. a new CuThreadsEventBridge

If developing Qt applications, consider using Qt's own event loop provided by QApplication. See the QThreadsEventBridgeFactory class in cumbia-qtcontrols module.
