Detailed Description

interface for a generic cumbia event designed to be generated in a secondary thread (CuActivity's thread - CuThread) and delivered into the main thread through an event loop.

This class is used internally.

The CuEventType enum defines some default event types used internally by the library and lets the user extend the types.

The pure virtual method getType must be reimplemented in subclasses to provide rtti for event type conversion at runtime.
