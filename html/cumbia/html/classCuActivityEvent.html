Detailed Description

Generic event to pass data to an activity.

Generic activity event that is posted from the main thread to the background thread.

Usages:

  • Cumbia::setActivityPeriod sends a CuTimeoutChangeEvent event
  • Cumbia::pauseActivity sends a CuPauseEvent
  • Cumbia::resumeActivity sends a CuResumeEvent
  • Cumbia::postEvent barely forwards a user defined CuActivityEvent to the specified activity
See also
Cumbia::postEvent (can be used by a client)
CuThread::postEvent (typically used internally by the library)
