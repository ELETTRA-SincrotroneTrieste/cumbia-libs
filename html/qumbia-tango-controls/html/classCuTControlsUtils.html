Replace wildcard arguments ($1, $2, ... $N) in tango sources with the command line arguments.

Parameters
thelist of the application arguments (qApp->arguments). The first one (the application name) will be ignored.
sthe source which wildcards will be replaced.
examples
  • source: $1/double_scalar args: ./bin/my_app 2 test/device/1 --> ./bin/my_app and 2 will be ignored. test/device/1 will replace $1
  • source: $2/double_scalar args: ./bin/app 1 –verbose test/dev/1 test/dev/c --> will be test/dev/c/double_scalar
