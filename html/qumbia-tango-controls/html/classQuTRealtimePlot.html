Detailed Description

A TPlotLightMarker specialized in managing real time commands.

This class inherits from TPLotLightMarker and is tailored to work with commands. Command syntax only is accepted by setSources, and the configuration dialog available through the right click menu allows the configuration of the real time parameters that caracterize the real time commands. The expected argout returned by the command is of type spectrum.

◆ setMode()

void QuTRealtimePlot::setMode ( int  m)

changes the mode of the configured sources

Note: call this after all the following methods, since setMode really changes the read mode of the plot (unsets the sources and sets them again), taking the parameters from the other properties.

modes 0: number of points mode: the configured number of points is read every refresh; 1: bunch1 to bunch N mode; 2: timestamp mode. The configuration parameters for the chosen mode (bunch1, bunch2, date1, date2) are Qt properties. They can be changed programmatically by the corresponding set... methods or by the right click dialog.
