void CuRndJsFunctionGen::generate ( CuData &  res)
virtual

Executes the function in the JavaScript file specified in the class constructor.

Parameters
respartially filled data that will store the result of the function in the "value" value.
Input arguments
The following input arguments will be passed to the JavaScript function, in this exact order. The arguments are stored in the options within the configure method.
  • size (the size of data to be generated: scalar or spectrum with the given size)
  • min the minimum value expected from the function
  • max the maximum value expected from the function
  • period the interval in milliseconds between subsequent function calls
  • call count, the number of times generate has been called so far
  • last result, the last valid value obtained from the previous call
Output from the JavaScript function
The result from the JavaScript function is stored into the value value within the input CuData argument passed as (non const) reference.

If an error occurs, err and msg values will be set accordingly
