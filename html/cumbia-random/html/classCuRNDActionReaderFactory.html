creates and returns a CuRNDActionReader, (that implements the CuRNDActionI interface)

Parameters
sa string with the name of the source
cta pointer to CumbiaRandom
Returns
a CuRNDActionReader, that implements the CuRNDActionI interface

The source and the CumbiaRandom reference are passed to the CuRNDActionReader If options have been set, normally through CuRNDActionReaderFactory::create, they are used to configure the CuRNDActionReader.

options
The recognised options are:
  • "period" an integer, in milliseconds, for the polling period. Converted with: CuVariant::toInt
  • "refresh_mode": an integer defining the Tango refresh mode. Converted with: CuVariant::toInt
note
Please use the CuTangoOptBuilder class rather than filling in the options manually.
