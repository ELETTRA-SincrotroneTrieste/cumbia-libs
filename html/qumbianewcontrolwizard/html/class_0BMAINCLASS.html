Detailed Description

$MAINCLASS$ is a reader that uses $SUPERCLASS$ to display values.

$MAINCLASS$ is a reader and a writer that uses $SUPERCLASS$ to display and write values.

// startdesc Connection is initiated with setSource. When new data arrives, it is displayed and the newData convenience signal is emitted.

getContext returns a pointer to the CuContext used as a delegate for the connection. // enddesc

// startdesc Connection is initiated with setSource. When new data arrives, it is displayed and the newData convenience signal is emitted. The write operation is executed on the endpoind with the same name as the source.

  • getContext implements the CuContextI interface and returns getOutputContext
  • getInputContext returns a pointer to the output CuContext used to read.
  • getOutputContext returns a pointer to the input CuContext used to write.

// enddesc
