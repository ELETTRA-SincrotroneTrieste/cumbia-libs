Cumbia http module

The cumbia http module provides an interface to send and receive data through http and https. The library relies on the canone3 server (caserver) that has been developed for the PWMA mobile and web interface:

  • The PWMA project

The cumbia http module has been developed in order to build applications for desktop and mobile devices able to interact with the Tango and EPICS control systems through the aforementioned canone server (PWMA).

CumbiaHttp is a CuHTTPActionListener in order to be notified when an action is finished, so that it can be unregistered and deleted.
