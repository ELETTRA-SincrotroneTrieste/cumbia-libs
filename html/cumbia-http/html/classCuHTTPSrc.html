◆ canMonitor()

bool CuHTTPSrc::canMonitor ( ) const

returns true if the source is suitable to be monitored

Returns
true if the source is a reader that can be monitored over time or if no helpers are installed. false if an installed helper's CuHttpSrcHelper_I::can_monitor returns negative for the given source.
Example
if CuHttpTangoSrcHelper is installed, canMonitor will return true if a source is an attribute or a command. They are indeed suitable to monitor values over time. The method returns false if the Tango helper detects a database data fetch related source, which shall be a one shot operation.
Default behaviour
If no helpers are installed, it is assumed that a source can monitor and true is returned
