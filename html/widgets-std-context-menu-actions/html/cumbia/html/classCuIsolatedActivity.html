Detailed Description

A convenient CuActivity designed to be executed only once.

This activity enables the flags CuActivity::CuADeleteOnExit and CuActivity::CuAUnregisterAfterExec and the CuIsolatedActivity::repeat method returns -1 in order to be executed once, automatically unregistered and disposed after its execution.

Two convenience methods unregisterOnExit and deleteOnExit are a shortcut to get the values of the aforementioned flags.
