qumbiaprojectwizard Documentation

qumbiaprojectwizard is an application to generate an empty cumbia project. It can also be used to migrate a simple QTango project to cumbia.

Create a new cumbia project

To start creating a cumbia project with qumbiaprojectwizard read Setting up a graphical application with cumbia and tango.

Command

cumbia new project
Migrate a project from QTango
To migrate a QTango project to cumbia read Porting a QTango application to cumbia-tango

Command

cumbia import
