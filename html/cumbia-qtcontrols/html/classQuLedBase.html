Detailed Description

A color led.

QuLedBase is a widget tailored to show true/false values. It has a color, a width and height that might be different from the container widget's width or height, depending on the scaleContents property being false or true respectively. Moreover, a led can be rectangular or elliptical (circular), according to the rectangular property. The color of the led can be shaded by means of a linear or rectangular gradient, and the start and stop values of the gradient can be fine tuned via the gradientStop and gradientStart properties. An integer between 0 and 255 can be specified to define the alphaChannel property, to add transparency to your led.

scaleContents property and gradientStop and gradientStart properties were introduced in QTango 4.x.
