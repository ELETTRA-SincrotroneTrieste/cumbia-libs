Detailed Description

convenience class to build a Qt QStringList from std::vector<std::string> or from a value stored in CuData

This class reduces the code that would be normally required to create a QStringList from a C++ std vector of strings. QuStringList is a Qt QStringList.

QuStringList's can be built from a std::vector<std::string>, a CuData bundle or a CuVariant type, if the data for which conversion is required is known to be either a vector of std::string or another type that can be converted to a string representation. Data extraction from CuVariant relies on the CuVariant::toStringVector method.

If the given CuData contains the given key, the associated value is converted to QuStringList.

Note
The CuVariant::toStringVector is used prior to QuStringList conversion. If the native type is not std::string, CuVariant::toStringVector tries to convert the elements into their string representation if possible. If conversion fails, the resulting QuStringList will be empty.
