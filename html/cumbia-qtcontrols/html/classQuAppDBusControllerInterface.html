Detailed Description

this class remotely controls an application registered with the DBus session bus

The implementation of this interface can:

  • find other applications registered with DBus by name and arguments
  • close an application matching the characteristics described in QuAppInfo
  • raise an application matching the characteristics described in QuAppInfo
