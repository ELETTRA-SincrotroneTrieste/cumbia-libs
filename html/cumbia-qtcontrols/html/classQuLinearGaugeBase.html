◆ drawBackgroundEnabled()

bool QuLinearGaugeBase::drawBackgroundEnabled ( ) const

returns true if a background is drawn, false otherwise

Returns
true the gauge draws a background with a radial gradient between the color associated the current value (see inErrorRange, inWarningRange, inNormalRange) and the color chosen with setBackgroundColor.

Three warning levels can be displayed by the gauge arc:

  • normal
  • warning
  • alarm (error)

Setting the drawBackgroundEnabled property to true enhances the detection of the current value quality.

double QuLinearGaugeBase::mapTo ( double  val,
const QRectF &  paint_area = QRectF() 
)

maps a value to a point along the longest side of the widget rectangle

Returns
a QPointF close to the arc of the gauge. How close it is to the arc is determined by tickLen: the point will be distant 1.2 * tickLen from the arc
Parameters
valthe value set on the widget
anglea pointer to a double that will store the angle, in degrees, from the 3 o'clock axis to the needle (if angle is not NULL)
radiusif greater than 0, the parameter is used to return a point with such distance from the center. Otherwise, radius is assumed to be equal to the paint area width divided by two.
paint_areaif valid, it is used for the calculation. Otherwise the paintArea method is used to recalculate the paint area
