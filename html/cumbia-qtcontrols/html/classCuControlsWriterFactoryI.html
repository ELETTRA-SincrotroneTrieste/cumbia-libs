Detailed Description

cumbia-qtcontrols writer factory. Creates an instance of a CuControlWriterA implementation.

  • CuTWriterFactory and CuEpWriterFactory implement CuControlsWriterFactoryI for Tango and Epics respectively;
  • CuTControlsWriter and CuEpControlsWriter implement CuControlsWriterA for Tango and Epics respectively.

Subclasses must implement the create, clone and setOptions pure virtual methods.
