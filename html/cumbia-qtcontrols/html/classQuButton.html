Detailed Description

A push button to write values or send commands to a target.

Derives from QPushButton and acts as a writer when clicked. Set up the link with setTarget. When clicked, the execute method is called, the (optional) input arguments are evaluated and the write is performed on the target.

See also md_src_cumbia_qtcontrols_widget_constructors documentation.
