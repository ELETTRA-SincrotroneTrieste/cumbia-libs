set the value to display on the label. Optionally check if the background color has been modified by the method

Parameters
vthe value as CuVariant.
background_modified,ifnot NULL, it will be set to
  • true if setValue changed the background of the label
  • false if setValue didn't change the background
The background of the label can be changed by set value if either
  • the type of data is boolean and the "trueColor" and "falseColor" properties are set to a valid QColor or
  • setEnumDisplay was used to associate integer values to a strings and a colors
Number formatting
If QuLabel::format is not empty, it is used to format the number. The format is forwarded to CuVariant::toString. If the latter method is called with an empty format, a default format is used within snprintf according to the data type in use.

See CuVariant::toString for further details.

Draws a decoration on the widget:

  • one pixel gray outer border
  • one pixel white border
  • borderWidth pixel with the color chosen for the border with the setDecoration method
  • one pixel inner white border
Note
the background color, set with the setDecoration method, is applied through the QLabel palette for the background role

sets the width of the border(s) drawn with the chosen color

Parameters
wthe desired width of the border

By default, two borders are drawn:

  • a light gray external border;
  • an green interior border if the read is successful, a red one otherwise

The interior border drawing can be turned off with setDrawInternalBorder
