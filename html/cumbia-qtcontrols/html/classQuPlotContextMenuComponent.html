Detailed Description

A QuPlotComponent managing the context menu for a plot.

This class, used by cumbia-qtcontrols plots, provides a context menu with default methods to:

  • print the plot
  • copy the plot as image
  • do a snapshot of the plot
  • save the data shown in the graph at that moment
  • configure the graph parameters (upper, lower bounds, curve colors, and so on)

The plot must have a QuWidgetContextMenuStrategyI installed. In this case, the execute method shows the menu populated according to the menu strategy under the mouse pointer.
