QuAppInfo constructor that accepts the full command line as provided by QApplication::arguments, including argv[0].

Parameters
_argsthe value returned by QApplication::arguments, argv[0] included.

Initializes the fields of the QuAppInfo structure. This will separate the application name and the command line options provided with the input parameter.

  • The exename method returns the argv[0]
  • args field (public) contains the command line options, without the argv[0]
  • path returns the application path, if provided, that is argv[0] - exename
  • argv0 returns the argv[0]
