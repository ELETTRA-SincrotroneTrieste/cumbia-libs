Detailed Description

template<typename T>
class QuVector< T >

convenience class that can be used to quickly convert a std::vector of a given type into a QVector of the same type

QuVector is a QVector and can be used to convert immediately std::vector based data into Qt QVector. Tedious and repetitive code writing is made unnecessary thanks to the versatile class constructors.

Example usage 1
Using QuVector you will be able to write
void Writer::hDataReady(const CuData& v)
{
QVector<double> readData = QuVector<double>(v[TTT::Value].toDoubleVector()); // v["value"]
// ...
}
