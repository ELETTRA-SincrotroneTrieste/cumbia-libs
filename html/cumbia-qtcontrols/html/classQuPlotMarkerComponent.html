QString QuPlotMarkerComponent::markerText ( const QuPlotBase * plot,
const QwtPlotCurve * closest_curve,
const int index )
virtual

returns a string containing a textual representation of the point on the curve the marker refers to

Parameters
plotthe pointer to the plot owning the marker
curvethe curve to which the marker refers
indexthe index of the sample on the curve that the marker refers to

This method can be reimplemented on a custom QuPlotMarkerComponent in order to provide a text different from the default one. An alternative to QuPlotMarkerComponent subclassing is offered by the QuPlotMarkerFormat_I interface: just implement its format method and install the object with setFormatter.

Default text
The default text contains *x: [x value]
[source name]:[source value]*
Note
If you need further customization for the component, reimplement the QuPlotMarkerComponent::update(const QuPlotBase *, QwtPlotCurve *, int ) method to provide a different placement strategy, text alignment and so on.

void QuPlotMarkerComponent::setFormatter ( QuPlotMarkerFormat_I * pmf)

Use a custom implementation of QuPlotMarkerFormat_I to format the marker text.

This offers an alternative to the reimplementation of QuPlotMarkerComponent::markerText in a subclass

QuPlotMarkerFormat_I subclasses shall implement the QuPlotMarkerFormat_I::format method to provide the marker's text.

Note
Ownership of the passed object is QuPlotMarkerComponent's
Subsequent calls delete previously installed QuPlotMarkerFormat_I
To remove an installed formatter, call this method with a nullptr argument
Parameters
pmfan implementation of the QuPlotMarkerFormat_I interface
