◆ getFreePropNam()

std::string TSource::getFreePropNam ( ) const

Returns whatever follows the "#" character, if present.

Returns
the subsctring of the source after the # character, if present. It should be a property name. An empty string is returned if no "#" character is found.
Examples
  • hokuto:20000/test/device/1/double_scalar#values (values attribute property)
  • test/de/1#dprop (dprop device property)
  • TangoTest#version (version class property)
  • #ObjectName#FreePropName (version free property)
See also
getFreePropObj
