CuTangoActionI * CuTangoReaderFactory::create ( const TSource & s,
CumbiaTango * ct ) const
virtual

creates and returns a CuTReader, (that implements the CuTangoActionI interface)

Parameters
sa string with the name of the source
cta pointer to CumbiaTango
Returns
a CuTReader, that implements the CuTangoActionI interface

The source and the CumbiaTango reference are passed to the CuTReader If options have been set, normally through CuTangoReaderFactory::create, they are used to configure the CuTReader.

options
The recognised options are:
  • TTT::Period an integer, in milliseconds, for the polling period. Converted with: CuVariant::toInt
  • "refresh_mode": an integer defining the Tango refresh mode. Converted with: CuVariant::toInt
note
Please use the CuTangoOptBuilder class rather than filling in the options manually.
